# Student-Data-Analysis

## 📌 Project Overview

This project focuses on analyzing student performance data using **Microsoft Excel** to uncover insights into academic performance, attendance, study habits, and overall student trends.

The analysis was performed using Excel tools such as:

* Pivot Tables
* Charts & Visualizations
* Conditional Formatting
* Data Cleaning Techniques
* Excel Formulas & Functions
* Dashboard Creation

The goal of this project was to transform raw student data into meaningful insights that can support better academic decision-making.

---

## 🗂️ Dataset Information

The dataset contains student-related information such as:

* Student ID
* Name
* Gender
* Age
* Class
* Subject
* Score
* Attendance Percentage
* Study Hours

---

## 🎯 Project Objectives

The main objectives of this analysis were to:

1. Analyze student academic performance.
2. Identify relationships between study hours and scores.
3. Examine attendance trends among students.
4. Compare performance across subjects and classes.
5. Build an interactive dashboard for quick insights.

---

## 🛠️ Tools Used

* **Microsoft Excel**

  * Pivot Tables
  * Pivot Charts
  * Conditional Formatting
  * Data Validation
  * Dashboard Design

---

## 📈 Key Insights

Some important insights discovered from the analysis include:

* Students with higher study hours generally achieved better scores.
* Attendance showed a positive impact on academic performance.
* Certain subjects recorded stronger overall performance than others.
* Performance differences were observed across gender and class categories.
* Visual dashboards made it easier to identify trends and outliers.
